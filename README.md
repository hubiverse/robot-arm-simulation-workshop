# Robotic Arm Simulation Workshop

This repository contains the code and resources for a robotic arm simulation workshop. The workshop is designed to teach participants how to simulate and control a robotic arm using Python and Gymnasium.

We will cover the following topics:
- Robotic arm simulation
- Robotic arm control
- Gymnasium
- Reinforcement learning (PPO and SAC)

Check out the `main.ipynb` notebook for more details.


## Setup
1. Clone the repository.
	```bash
	git clone https://github.com/hubiverse/robot-arm-simulation-workshop.git
	```
2. Install `uv` using the [installation instructions](https://docs.astral.sh/uv/getting-started/installation/).
3. Create a virtual environment and install the dependencies.
	```bash
	cd robot-arm-simulation-workshop
	```
    ```bash
     uv sync
    ```
   
4. Run the `main.ipynb` notebook to get started with the workshop.
	```bash
    uv run jupyter lab
    ```
5. Enjoy!
